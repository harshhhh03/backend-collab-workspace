# Backend Collaborative Workspace (FastAPI)

## Overview
This project is a backend service for a real-time collaborative workspace, built as part of the PurpleMerit Backend Developer Assessment.

It demonstrates secure API design, real-time communication, and cloud-ready deployment using a clean and minimal architecture.

---

## Tech Stack
- Backend Framework: FastAPI (Python)
- API Documentation: Swagger (OpenAPI)
- Real-Time Communication: WebSockets
- Deployment: Render
- Version Control: GitHub

---

## Features Implemented
- Health check API
- WebSocket-based real-time communication (basic)
- OpenAPI/Swagger documentation
- Cloud deployment with live URL
